開発言語/Python 3.10.5
pip/flask 3.0.2

【注意事項】
本アプリは現在テスト開発段階です。
予期せぬ不具合がある可能性がありますので、発見されましたらご連絡ください。

ファイルの中にcopyとついたものが複数存在します。
全て開発途中のバックアップや別案のプログラム等ですので、無視してください。

試験問題の内容は全て questions.json ファイルにて管理しています。
問題を編集する時はこちらのファイルからお願いします。


【動作について】
起動は app.py からお願いします。
index.htmlにて「開始」ボタンを押下すると出題ページ（questions.html）へ遷移します。
問題ごとに選択肢の中から答えと思うものを選択します。（終了するまで自由に変更可能です。）
ページ右上に問題数にあわせた解答率を表示しています。
ヘッダー部分（タイトルの真下）に、解答率に応じて100%に伸びていくゲージを実装しています。
ページ右下にはTOPへ戻るためのボタンを用意しています。
問題を解き終わったら、ページ最下部にある「解答を終了」ボタンを押下します。
終了するとTOPへ戻り、採点結果と正解率が表示され、各問題ごとに正解だったか不正解だったかを表示します。
終了時には解答率のゲージが正解率のゲージへと変わり、100%に向かって再度伸びていきます。

※テストを円滑にするため問題は10問にしていますが、問題数は都度変更可能です。


【jsonファイル内の解説】（例として1問目を元にしています。）
≪問題ごとに振られるID≫
"id": 1,

≪問題本文≫
"question": "Pythonの創設者は誰ですか？",

≪解答の選択肢≫
"choices": ["ケン・トンプソン", "デニス・リッチー", "ジェームズ・ゴスリン", "グイド・ヴァン・ロッサム", "スティーブ・ジョブズ"],

≪答えとなる選択肢の番号（この問題の場合は"グイド・ヴァン・ロッサム"）≫
"answer": 4,

≪正解だった場合の点数≫
"points": 2

この一連のコードをコピーしてIDを振れば問題を追加、コードを消せば減らすことが可能です。


【追加機能】2024/04/23
index.htmlに「学生番号」入力フォームを追加しました。
任意の7桁の番号を入力して開始ボタンを押すと、questions.htmlのヘッダー部に当番号が表示されます。
（番号を入力しないと開始することが出来ません。）

index.htmlの「開始」ボタン、及びquestions.htmlの「解答を終了」ボタンを押すと、確認のダイアログが表示されるようになりました。

questions.htmlのヘッダー部に、「印刷」ボタンを追加しました。（現在は「PDFで保存」ボタンも配置しています。）
※「PDFで保存」ボタンはテスト機能のため、不具合が生じる可能性があります。
　問題数が10問程度であれば動作しますが、50問程になると処理が出来なかった現象を確認済です。
　現在の仕様では、Google chromeブラウザにて「印刷」ボタンを押して、開いたウィンドウで「送信先」を「PDFに保存」にして出力した方が良いです。
